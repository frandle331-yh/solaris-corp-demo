# SOLARIS — WordPressコーポレートサイト（制作実績用デモ）

架空の一般企業「ソラリス株式会社」を題材にした、コーポレートサイトの制作実績デモです。
「デザインリニューアル」＋「NEWSページの追加（クライアント自身が記事を投稿・更新できる仕様）」という、
実際のWordPress改修案件でよくあるご依頼を想定して制作しています。

**デモURL**: https://<ユーザー名>.github.io/solaris-corp-demo/

## 構成ページ

- トップページ（ヒーロー / 事業内容 / 導入事例 / 会社紹介 / ご相談の流れ / お知らせ / CTA / フッター）
- お知らせ一覧（カテゴリー絞り込み・ページ送り）
- お知らせ記事詳細

## デザイン・実装のポイント

- 写真主体のモダンなコーポレートデザイン（ファーストビュー全面写真＋透過ヘッダー）
- 導入事例セクションは「削減率・時間」など**数字で成果が伝わる**構成
- スクロール連動のフェードインアニメーション（`IntersectionObserver`、reduced-motion対応）
- モバイル対応：ハンバーガー→ドロワーメニュー、ブレイクポイントごとの最適化
- CTA（無料相談）への導線をヘッダー・各セクション・ページ末尾に配置

## 技術要素

- HTML / CSS / JavaScript（レスポンシブ対応：PC・タブレット・スマホ）
- WordPress子テーマ一式を同梱（`wp-theme/`）
  - お知らせ用カスタム投稿タイプ + カテゴリーを登録し、**管理画面から専門知識なしで記事を投稿・更新できる仕様**
  - 既存サイトを壊さない子テーマ構成
- 画像は `assets/img/` に同梱（写真素材：Unsplash（Unsplash License・商用利用可））

## ディレクトリ

```
index.html / news.html / news-article.html   … 静的デモ（GitHub Pages公開用）
assets/style.css                              … 共有スタイル
assets/script.js                              … ヘッダー・ドロワー・アニメーション
assets/img/                                   … 写真素材（Unsplash）
wp-theme/solaris-child/                        … WordPress子テーマ一式（実装コード）
```
